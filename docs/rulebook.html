ゲームの流れ

1. メインメニュー

ゲームを開始するには、Startが赤く囲ってある状態でボタン1を押します。

2. コントローラー接続画面

コントローラーの接続を確認する画面です。「接続済み」と表示されていればボタン3で次の画面に移ってください。

3. キャラクター選択

プレイするキャラクターを選択します。キャラクターごとにステータスや必殺技が異なるので、お好きなキャラクターを選んでください。
キャラクターの変更はジョイスティックを左右に動かして行います。
キャラクターが決まったらボタン3で次の画面に進みます。

4. 戦闘待機

戦闘が始まる前の待機画面です。準備ができたらボタン1で戦闘開始してください。押さなくても10秒で自動開始します。

5. 戦闘中

戦闘が始まります。相手の体力を0にするか、制限時間内に体力の割合が多い方がその試合の勝者となります。1プレイあたり3ラウンドあります。
画面一番上の緑のバーが体力、その下の青いバーが必殺技ゲージ、一番下の赤いバーが遠距離攻撃ゲージです。
ジャンプや攻撃をして、必殺技ゲージが満タンになると赤と青のグラデーションになります。
遠距離攻撃ゲージが満タンになると赤く光ります。

6. リザルト

3ラウンド終了後、勝敗が表示されます。

  • 各ラウンドと総合の勝者が表示されます
  • シングルプレイの場合はスコアも表示されます
  • スコアは基本的にボットに与えたダメージで、頭に攻撃があたった際のボーナス点を加えて算出されます。
  • 結果を確認したらボタン1を押してメインメニューに戻ってください

🎮 ボタン

ボタン2 戻る

種類 操作 説明
基本攻撃 ボタン2 パンチ - 素早い近距離攻撃。連続で繰り出せる
ボタン3 キック - パンチより威力の高い近距離攻撃
特殊攻撃 左/右 → ボタン3 バックキック - より強力でリーチの長い攻撃
ボタン4 遠距離攻撃 - ゲージを消費して炎の矢を飛ばす(ゲージ満タン時のみ)
必殺技 ボタン1 キャラクター固有の強力な技(ゲージ満タン時のみ)
移動・回避 左/右 移動
上 (→ ボタン3) ジャンプ - ボタン3でジャンプキック
左 → 上 (→ ボタン3) 左ジャンプ - 前方ジャンプの場合、ボタン3でジャンプキック
右 → 上 (→ ボタン3) 右ジャンプ - 前方ジャンプの場合、ボタン3でジャンプキック
下 しゃがむ - 防御力が増加
左/右 → 下 ロール - 遠距離攻撃を回避できる
